CampusCoins Setup Guide

Initial Setup Instructions

1. Deploy the Contract

truffle migrate --reset

This will deploy the CampusCoin contract and automatically set up the roles.

2. Verify Roles

truffle exec check_roles.js

This will check if the roles are properly set up in the blockchain.

3. Manual Role Setup (if needed)

truffle exec setup_roles.js

If roles were not set up properly during migration, this script will set them manually.

4. Update Contract Address

python deploy.py

This will update the contract address in app.js with the newly deployed address.

5. Start the Application

python app.py

This will start the Flask web application.
